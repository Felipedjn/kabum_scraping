Script web scraping de produtos da Kabum. Construído em Python 3.10.

# Instalação

- pip install requests
- pip install bs4
- pip install re
- pip install pandas
- pip install math

## Como pesquisar um item

No campo "pesquisa" adicione o item que você quer puxar
